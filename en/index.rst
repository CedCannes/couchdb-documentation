Welcome to Doctrine CouchDB documentation
=========================================

The Doctrine project offers PHP libraries that handle data persistence.
Doctrine CouchDB maps Apache CouchDB documents to PHP objects. It also
ships with a thin wrapper around the CouchDB HTTP API.
